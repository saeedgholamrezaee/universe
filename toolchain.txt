Toolchain یعنی چه؟

Toolchain = مجموعه ابزارهای لازم برای ساخت (build) یک برنامه C/C++

این ابزارها شامل:

کامپایلر (gcc / clang / msvc)

لینک‌کننده (ld / lld / link.exe)

استاندارد کتابخانه C/C++ (libstdc++ / libc++ / msvc STL)

مسیرهای include و lib

تنظیمات وابستگی (dependencies)

تنظیمات ABI، architecture و build type
